# PHP Plugin for Project Client

1. Install a plugin
```
$ npm install -g project-client-plugin-php
```

2. Configure plugins of projects in ~/.project-client.json to php. For example,
```
{
    "laravel": {
        "plugins": "php",
        "repository": "https://github.com/at-ishikawa/laravel.git"
    }
}
```
